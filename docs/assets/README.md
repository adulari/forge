# README assets

## Demo GIFs

`forge-demo.gif` (the README hero) and `demo-*.gif` (the per-feature "See it in action" clips)
are recorded by **`scripts/demo/record.sh`** — reproducible vhs tapes running a fully offline
`--mock` session in an isolated stage. See `scripts/demo/README.md` for requirements and how the
staging works. Don't edit the GIFs by hand; change the tapes and re-record.
